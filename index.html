Piazza Post Classifier

Class Project - EECS 280

Private Repo

A program created using C++ that automatically identifies the subject of posts on Piazza, a website where students can ask instructors questions. A probabilistic model of how piazza posts are composed was built using Conditional Probability, Bayes Theorem, Bag of Words Model, and Logarithmic Probabilities. The program was trained on 11,400 posts from students in the 2014-2015 semesters and tested on 2988 posts from the 2016 semester. The algorithm was able to classify 88% (2602/2988) of posts into correct folders.

Image Resizing

Class Project - EECS 280

Private Repo

In this project, I created a program to resize images in C++ using a seam carving algorithm which is a technique to resize images in a way that changes aspect ratio but does not distort image. The algorithm works by finding and removing "seams" in the image that passes through the least important pixels.
